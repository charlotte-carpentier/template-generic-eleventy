## How to Contribute

1. **Follow conventions** - See [Quick Reference](../02-using/08-quick-reference.md) for naming and code standards
2. **Test locally** - Run `npm start` and verify changes before committing
3. **Update documentation** - Document new features or changes in relevant MD files
4. **Commit format** - Use emoji commit pattern (see Quick Reference)
5. **Add yourself** - Update Core Team section below with your contribution
